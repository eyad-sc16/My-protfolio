    Tools

    The techniques I used
    GSAP

    Smooth motion and transitions that make the site feel alive

    GSAP stands for GreenSock Animation Platform, a powerful JavaScript library for creating high-performance animations. GSAP is a robust JavaScript library widely used by web developers and designers to create complex animations with precision and efficiency.

    ScrollTrigger

    Scroll-triggered animations that keep visitors engaged

    ScrollTrigger is a GSAP plugin that links animations to scroll position. It enables pinning, scrubbing, and revealing content as users move through the page with smooth, controlled timing.

    SplitType

    A silky, lag-free browsing experience

    SplitType is a JavaScript library that splits HTML text into individual characters, words, or lines. It makes typography animations easier with staggered effects and creative text reveals.

    Custom Properties

    Fully responsive design that looks great on any screen or device

    CSS Custom Properties let you store reusable values like colors, spacing, and fonts in one place. They make themes easy to maintain and update across the entire site.

    Grid & Flexbox

    Custom visual identity, not a copy-paste template

    Grid and Flexbox are modern CSS layout systems for building responsive designs. Flexbox handles one-dimensional layouts, while Grid gives precise control over two-dimensional page structure.

    Clip-path

    Easy to update themes and colors down the line

    Clip-path is a CSS property that clips an element to a specific shape. It enables creative visual effects like diagonal cuts, circles, and custom masks without extra images.
